# PLAN DE NEGOCIO INTEGRAL: STARTUP VORA

## 1. MODELO DE NEGOCIO
- **Formato**: Dropshipping Premium (Stock en Cúcuta, despacho desde Táchira).
- **Propuesta de Valor**: "El lujo americano antes de que llegue América". Calidad de $130 por $80, con envío gratis.
- **Target**: Público Urbano (Sneakers) y Mujer Gala (Tacones) en las 5 ciudades principales de Venezuela.

## 2. ESTRATEGIA DE PRECIOS
- **Sneakers Premium**: Costo $40 -> Venta $85 (Utilidad $45).
- **Heels/Gala**: Costo $20 -> Venta $45 (Utilidad $25).
*Nota: El precio de venta incluye el envío nacional ($5).*

## 3. EMBUDO DE VENTAS (SALES FUNNEL)
- **Tráfico**: Marketplace de Facebook (Saturación geográfica) + Google Business + TikTok Orgánico.
- **Conversión**: Landing Page en Antigravity (Autoridad) -> Botón de WhatsApp.
- **Cierre**: Agente de IA configurado por Antigravity para responder dudas de tallas, confirmar disponibilidad y enviar datos de pago (Binance, Zelle, Pago Móvil).

## 4. INFRAESTRUCTURA TÉCNICA
- **Repositorio Central**: GitHub (Gestionado por Antigravity).
- **IA Creativa**: Generación de modelos hiperrealistas y escenarios de lujo para el producto.
- **Automatización**: Sincronización de inventario automática entre el repositorio y el catálogo de WhatsApp.
